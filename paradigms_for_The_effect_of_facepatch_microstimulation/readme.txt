This directory contains the kofiko paradigm file for each figure panel in:
Moeller, Sebastian, Trinity Crapse, Le Chang, and Doris Y. Tsao. 
‘The Effect of Face Patch Microstimulation on Perception of Faces and Objects’. 
Nature Neuroscience 20, no. 5 (May 2017): 743–52. https://doi.org/10.1038/nn.4527.

This can be used to figure out the exact images used for each experiment.
